# My Static Site

This is a simple static website project that includes an HTML file, a CSS file for styling, and a folder for images.

## Project Structure

```
my-static-site
├── index.html       # Main HTML document for the static site
├── style.css        # Styles for the HTML document
├── img              # Folder to store image files
└── README.md        # Documentation for the project
```

## Getting Started

To view the static site, open the `index.html` file in your web browser.

## Customization

You can customize the styles by editing the `style.css` file. Add your images to the `img/` folder and reference them in the `index.html` file as needed.
